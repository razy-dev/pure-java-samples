Chain of Responsibility

  • 연속된 기능 클래스가 각자 자신이 처리할 수 있는 요청만 처리

  • eg> Spring Security Filter Chain

next

Command

  • 요청을 객체로 캡슐화, 요청에 기능 추가

Mediator

  • 중간객체(Mediator)가 여러 객체들의 상호작용 간략화

Colleague, 상호관계, 상호작용

Memento

  • 상태 정보를 저장

rollback, undo

Observer

  • 객체의 상태가 변할 때 다른 객체들에게 변화를 통지

notify, update

State

Context, State
