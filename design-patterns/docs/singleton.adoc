= Singleton Pattern

== Descriptions
인스턴스를 하나만 만들어 사용하기 위한 패턴이다.

즉 생성자의 호출이 반복적으로 이뤄져도 실제로 생성되는 객체는 최초 생성된 객체를 반환해주는 것이다.
