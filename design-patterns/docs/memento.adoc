= Memento Pattern

== Descriptions
객체를 이전 상태로 되돌릴 수 있는 기능을 제공

오리지네이터(originator):: 상태를 저장하고 복원할 객체의 주체이다.
케어테이커(caretaker):: 오리지네이터의 저장과 복원을 제어한다.
메멘토(memento):: 오리지네이터의 상태를 저장한다.

* 메멘토 객체 자신은 불투명 자료형(케어테이커는 변경할 수 없거나 변경해서는 안 되는)이다.
